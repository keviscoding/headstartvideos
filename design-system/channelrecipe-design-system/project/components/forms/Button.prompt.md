**Button** — rounded CTA. Primary is violet with a soft violet lift shadow; secondary is an ink outline; ghost is text-only; subtle is a violet tint. Label with the exact action.

```jsx
<Button variant="primary" size="lg">Make your first video free</Button>
<Button variant="secondary">See a recipe</Button>
<Button variant="primary" theme="dark" full>Cook video</Button>
```

Sizes `sm|md|lg`; `full` stretches to container; pass `theme="dark"` inside the app.
