**StatusChip** — mono, uppercase status semantic. Violet `new` for the weekly drop; green `proven` carries a tiny dashboard spark; `monetized`, `warn`, `error`, `neutral` for the rest. Green is status only, never decoration.

```jsx
<StatusChip kind="new">New</StatusChip>
<StatusChip kind="proven">Proven</StatusChip>
<StatusChip kind="monetized">Monetized</StatusChip>
```
